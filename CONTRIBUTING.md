# Contributing to Image Squisher

Thank you for your interest in contributing!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/image-squisher.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Test your changes
6. Submit a pull request

## Development Setup

```bash
# Set up the project
./setup.sh
source venv/bin/activate

# Make your changes to the code

# Test your changes
python main.py /path/to/test/images
```

## Code Style

- Follow PEP 8 style guidelines
- Use type hints where appropriate
- Add docstrings to functions
- Keep functions focused and small

## Testing

Before submitting a PR, please:
- Test with various image formats
- Test with both small and large image sets
- Verify error handling works correctly
- Check that logging works as expected

## Pull Request Process

1. Update the README.md if needed
2. Ensure all code follows the style guidelines
3. Test your changes thoroughly
4. Submit the PR with a clear description of changes

## Questions?

Feel free to open an issue for questions or discussions!
